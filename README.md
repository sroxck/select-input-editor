# Select-input-editor
模仿 `tally.so` 编写的一个可以输入的文本框

## 已完成功能
1. 可以自由输入文本
2. 回车新增文本框
3. 输入 '/' 启动组件面板 
4. 输入 '/组件名' 快速选择组件,回车生成该组件
5. 输入 '/' 后通过方向键快速选择组件,回车生成该组件

## 准备实现
1. 输入框左侧操作按钮,新增,删除,配置等功能实现
2. 生成的组件绑定数据对象
3. 在生成的表单组件中可以继续通过'/'来生成
4. 生成的组件支持拖拽换位
